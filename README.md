# Kolorowe przyciski nawigacji
### Rozmiar przycisków jest w pełni uzależniony od niestandardowej właściwości:
```css
--bar-size
```
### Kolor komponentu można kontrolować za pośrednictwem nazw klas:
* red
* orange
* green
* blue (domyślne ustawienie komponentu)
* purple
* pink
